# Coursework Tools — Prodigy Solutions

Browser-based tools for inmate education programs. No install required — open the site and pick a tool.

**Live:** https://wynnchance.github.io/reward-tracker/

## Tools

- **Reward Tracker** — `tracker.html`
  Track Kolibri coursework time across facilities and manage coursework rewards, with audit exports, history, and optional GitHub Gist sync.

- **Certificate Generator** — `certificate.html`
  Generate a certificate of completion and a course-by-course transcript for any learner from Kolibri log exports, ready to print or save as PDF.

## How it works

Both tools run entirely in your browser using the CSV files exported from Kolibri (Facility → Data). No data leaves the page, except the Reward Tracker's optional Gist sync.
